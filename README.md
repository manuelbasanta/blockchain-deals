## Run Locally


Clone the project

```bash
  git clone https://github.com/manuelbasanta/blockchain-deals.git
```

Go to the project directory

```bash
  cd blockchain-deals
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
